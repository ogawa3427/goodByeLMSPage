プライバシーポリシー / Privacy Policy

最終更新 / Last updated: 2026-04-11

「Good Bye LMS Page」（以下「本拡張機能」）は、金沢大学の学習管理システム（LMS）へのアクセスを補助することを目的とした非公式の Chrome / Firefox 拡張機能です。

1. 収集するデータ / Data Collected

本拡張機能は以下の情報をお使いのデバイス上（chrome.storage.local 相当）にのみ保存します。

  • 金沢大学 LMS の講義 URL および講義名（学務情報サービスの履修時間割ページから取得）
  • 拡張機能の動作設定（表示設定など）

本拡張機能はユーザーの個人情報（学籍番号・パスワード等）を収集・外部送信しません。取得した講義情報もお使いのブラウザ内にのみ保持され、外部のサーバーへ送信されることはありません。

2. 外部への通信 / External Communication

本拡張機能は以下のドメインにのみアクセスします。

  • acanthus.cis.kanazawa-u.ac.jp – アカンサスポータルへのページ操作
  • eduweb.sta.kanazawa-u.ac.jp – 学務情報サービスからの履修データ取得
  • api.github.com – 拡張機能の更新情報確認（バージョン番号のみ）

これらへの通信は HTTPS で行われます。ユーザーの個人情報はいかなるリクエストにも含まれません。

3. 第三者への提供 / Disclosure to Third Parties

本拡張機能は、収集したいかなるデータも第三者に提供しません。

4. データの削除 / Data Deletion

ブラウザの拡張機能管理画面から本拡張機能をアンインストールすると、保存されたすべてのデータが削除されます。

5. お問い合わせ / Contact

プライバシーに関するご質問は GitHub Issues よりご連絡ください。

Privacy Policy (English)

"Good Bye LMS Page" is an unofficial Chrome/Firefox extension designed to assist access to Kanazawa University's Learning Management System (LMS).

1. Data Collected

The extension stores the following data locally on your device only (via chrome.storage.local):

  • LMS course URLs and course names (retrieved from the academic schedule page)
  • Extension settings (display preferences, etc.)

No personal information (student ID, password, etc.) is collected or transmitted externally. All retrieved course data remains in your browser and is never sent to any external server.

2. External Communication

The extension communicates only with the following domains:

  • acanthus.cis.kanazawa-u.ac.jp – page operations on the Acanthus portal
  • eduweb.sta.kanazawa-u.ac.jp – retrieving course registration data
  • api.github.com – checking for extension updates (version number only)

All connections are made over HTTPS. No personal information is included in any request.

3. Disclosure to Third Parties

The extension does not share any collected data with third parties.

4. Data Deletion

Uninstalling the extension via your browser's extension management page will remove all stored data.

5. Contact

For privacy-related questions, please open an issue on GitHub Issues.
